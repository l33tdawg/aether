Key Innovation

Shared Dismissals Kill Duplicate Findings

When one agent dismisses a concern with evidence, every subsequent agent knows. No more re-flagging the same non-issue across passes.

✕ Before — v5.0

Agents Work in Isolation

Pass 3 analyzes reentrancy, decides it’s protected by a nonReentrant modifier, and dismisses it. Pass 5 has no memory of this — re-flags the same reentrancy concern. Pass 4 does too.

~40% duplicate findings
✓ After — v6.0

Agents Share Everything

Pass 3 stores a dismissal to SAGE: "reentrancy: protected by nonReentrant on L142". Pass 5 reads it — skips reentrancy entirely, focuses on novel attack vectors instead.

0 duplicate findings

Get Running

Two Commands. That’s It.

SAGE runs as a local Docker service. Aether connects automatically.

terminal 
# Clone and set up
$ git clone https://github.com/l33tdawg/aether
$ cd aether && python setup.py

# Start SAGE institutional memory
$ docker compose up -d

# Launch Aether
$ python aether.py
